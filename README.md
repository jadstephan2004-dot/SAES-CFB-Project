# Simplified AES (S-AES) with CFB Mode
This project implements S-AES encryption and decryption in Cipher Feedback (CFB) mode.

## Files
- `saes_cfb.py`: Contains the S-AES implementation and CFB logic.
- `demo_files.py`: Generates encrypted text and a chessboard image.
- `brute_force.py`: Demonstrates a brute-force attack on a 16-bit key.
